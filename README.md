# Nicky Myny

Next.js (App Router) site for [nickymyny.com](https://nickymyny.com).

## Develop

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
pnpm build
pnpm start
```

## Env

No env vars required today. Inquire and newsletter forms open a prefilled `mailto:` to
`artist@nickymyny.com`. Add a mail provider later if server-side sending is needed.

## Deploy

Designed for Vercel (`pnpm build`).
